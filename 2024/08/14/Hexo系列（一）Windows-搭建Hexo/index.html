发表于2024-08-14|更新于2024-09-26
|阅读量:

如果你想要拥有自己的博客，如果你暂时不想斥资购买服务器、数据库、域名，那你只需要一个 github 账号，就可以 跟着我从 0 到 1 使用 Hexo 框架和 Butterfly 主题搭建一个属于自己的博客！

什么是 Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他标记语言）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Hexo 做了什么

当我们执行命令 hexo generate 的时候，hexo 会将博客内容生成静态 HTML 文件，写入到 moon/public 文件夹， hexo deploy 命令 会将其部署到 GitHub Pages, 还记得我们前面在 _config.yml配置的 Github 仓库信息吗？

发布第一篇文章

我们有了自己的博客，那如何发布文章就是我们下一步要做的事情！

在 moon 目录下，我们使用命令 hexo new post "我的第一篇文章", 这样在 source\_posts 目录下就会生成一个 我的第一篇文章.md 的 Markdown 文件，你可以编辑它，开启你的写作之旅。
