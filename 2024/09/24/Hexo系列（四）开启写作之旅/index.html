Hexo系列（四）开启写作之旅

发表于2024-09-24|更新于2024-09-26
|阅读量:

之前在 使用 Hexo 从 0 到 1 搭建博客中我们已经发布了第一篇文章，但是基于 Hexo 写博客文章的技巧还有很多，让我们一点点的学起来，让工具越来越顺手！

首先，一次完整的发布文章的流程应该是：

  1. 使用命令创建一个文章 markdown 文件
  2. 在 markdown 中编写文章
  3. 本地部署预览
  4. 远程部署发布

封面尺寸

Tips

有一点需要注意的是，我们在给图片命名的时候，名字中间不能出现空格，否则引入的时候会与 markdown 语法冲突，出现图片无法引入或格式错误的问题。
